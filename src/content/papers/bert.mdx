<StageSection id="intuition" title="① Intuition">

Before BERT, the dominant language model paradigm was **unidirectional**: GPT reads tokens strictly left-to-right (causal masking), meaning when predicting or encoding position i, the model only sees tokens to the left. This is natural for generation, but it's a significant limitation for understanding — knowing both the left and right context of a word is crucial for tasks like question answering and named entity recognition.

BERT's key insight: instead of predicting the next token, **mask out random tokens and predict them using the full surrounding context** — bidirectional attention in both directions simultaneously. This is called **Masked Language Modeling (MLM)**. On top of that, BERT adds **Next Sentence Prediction (NSP)**: given two sentences A and B, predict whether B is the actual next sentence in the corpus or a random one. Together, these two pretraining objectives allow BERT to build rich, context-aware representations of text.

<PredictThenReveal client:load predictPrompt="Why can't GPT-style (left-to-right) pretraining learn bidirectional context, even if you stack many layers?">
  Causal masking is applied at the attention layer: each token's attention is literally blocked from attending to any future position. No amount of depth overcomes this architectural constraint — the information never enters the computation graph. BERT removes causal masking entirely during pretraining, letting every token attend to every other token.
</PredictThenReveal>

<StageSection id="math" title="② Math">

BERT's pretraining loss is the sum of two cross-entropy terms. First, **MLM loss** sums over the set M of masked positions:

<Math display expr="L_\text{MLM} = -\sum_{i \in M} \log p(x_i \mid x_{\setminus M})" />

Here the conditioning context is the full input with masked positions replaced — the model predicts the original token x_i using all non-masked context. Second, **NSP loss** uses the [CLS] token's pooled representation:

<Math display expr="L_\text{NSP} = -\log p(\text{IsNext} \mid [\text{CLS}])" />

The total pretraining loss is simply their sum:

<Math display expr="L = L_\text{MLM} + L_\text{NSP}" />

  hints={[

    "For each selected position, draw a uniform random float. If < 0.8: replace with mask_token_id. If < 0.9: replace with np.random.randint(0, vocab_size). Otherwise: leave unchanged.",
    "Final structure: selected = np.random.rand(n) < 0.15. For i in np.where(selected)[0]: r = np.random.rand(); if r < 0.8: out[i] = mask_token_id; elif r < 0.9: out[i] = randint(vocab_size). Return out, selected.",
  ]}
  solutionCode={`import numpy as np

def bert_masking(tokens, mask_token_id, vocab_size, mask_prob=0.15, seed=None):
    rng = np.random.default_rng(seed)
    out = tokens.copy()
    selected = rng.random(len(tokens)) < mask_prob
    for i in np.where(selected)[0]:
        r = rng.random()
        if r < 0.80:
            out[i] = mask_token_id
        elif r < 0.90:
            out[i] = rng.integers(0, vocab_size)
        # else: leave unchanged
    return out, selected`}
