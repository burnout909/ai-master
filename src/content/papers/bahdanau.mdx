<StageSection id="intuition" title="① Intuition">

Before attention, seq2seq models compressed an entire source sentence into a single fixed-size vector **C** — the encoder's final hidden state. The decoder then generated every target token from that one vector alone. For short sentences this works. For long ones, the model must stuff "The European Parliament has rejected the proposal to postpone the amendment to the Tobacco Products Directive" into a 256-dimensional vector. Early words get crushed out by later ones. Translation quality collapses.

**Bahdanau's fix:** instead of a static C, give the decoder a *dynamic* context vector <Math expr="c_i" /> at each output step *i*. The decoder computes a soft alignment weight <Math expr="\alpha_{ij}" /> over every encoder hidden state <Math expr="h_j" />, then takes a weighted sum. The decoder "peeks" at the full source sequence each step, focusing on the parts most relevant to the current output word.

  **'étudiant'** — the direct semantic match. Soft alignment lets the decoder assign near-1 weight to 'étudiant' and near-0 to everything else when generating 'student'. This is the core insight: attention is a learned, differentiable lookup over the source sequence.

<StageSection id="math" title="② Math">

The encoder is a **bidirectional RNN**. For source token j, forward and backward hidden states are concatenated:

<Math display expr="h_j = \left[\overrightarrow{h}_j\,;\,\overleftarrow{h}_j\right]" />

At decoder step i, an **alignment score** measures how well source position j matches the current decoder state <Math expr="s_{i-1}" />:

<Math display expr="e_{ij} = v_a^\top \tanh\!\left(W_a\,s_{i-1} + U_a\,h_j\right)" />

This is the **additive (Bahdanau) score function** — a small one-hidden-layer MLP with learned weight matrices <Math expr="W_a, U_a" /> and vector <Math expr="v_a" />.

Softmax over all source positions gives a **probability distribution**:

<Math display expr="\alpha_{ij} = \frac{\exp(e_{ij})}{\sum_{k=1}^{T_x} \exp(e_{ik})}" />

The **context vector** is the resulting convex combination:

<Math display expr="c_i = \sum_{j=1}^{T_x} \alpha_{ij}\, h_j" />

Implement the Bahdanau additive attention score and softmax weighting in NumPy. Given decoder state `s_prev`, encoder states `H`, and projection matrices `W_a`, `U_a`, `v_a`, return the attention weights `alpha` (shape `(T_enc,)`) and context vector `c` (shape `(d_h,)`).

<HintFade client:load
  hints={[
    "Compute scores e_j = v_a · tanh(W_a @ s_prev + U_a @ h_j) for every j. You can vectorise by projecting all H at once: (U_a @ H.T).T gives shape (T, d_a).",
    "Add W_a @ s_prev (shape d_a) to each row of the H projection, then apply np.tanh element-wise. Finally dot with v_a to get a (T,) vector of scores.",
    "For numerically stable softmax: subtract np.max(scores) before np.exp, then divide by the sum. Context is alpha @ H.",
  ]}
  solutionCode={`import numpy as np

def softmax(x):
    x = x - np.max(x)
    e = np.exp(x)
    return e / e.sum()

def bahdanau_attention(s_prev, H, W_a, U_a, v_a):
    # s_prev: (d_s,)  H: (T, d_h)
    proj_s = W_a @ s_prev          # (d_a,)
    proj_H = (U_a @ H.T).T         # (T, d_a)
    scores = np.tanh(proj_H + proj_s) @ v_a   # (T,)
    alpha = softmax(scores)        # (T,)
    c = alpha @ H                  # (d_h,)
    return alpha, c`}
