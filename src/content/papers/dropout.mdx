<StageSection id="intuition" title="① Intuition">

When you train a large network long enough on limited data, neurons start to collude. Unit A fires whenever unit B fires; together they memorize a specific pattern in the training set — a texture, a noise artifact, an ID of one example. This is **co-adaptation**: neurons rely on each other rather than each learning something independently useful. The result is a network that looks brilliant on training data and flails on anything new.

**Dropout's insight:** randomly silence a different random subset of neurons on every forward pass. Now unit A can never count on B being present. Each unit is forced to be useful on its own. At test time every neuron is active, but we scale outputs down by `keep_prob` to match expected training activations.

<StageSection id="math" title="② Math">

Let <Math expr="p" /> be the **keep probability**. During training, each unit <Math expr="h_i" /> is multiplied by a Bernoulli random variable <Math expr="r_i \sim \text{Bernoulli}(p)" />:

<Math display expr="\tilde{h}_i = r_i \cdot h_i, \quad r_i \sim \text{Bernoulli}(p)" />

At test time all units are active. To match the expected activation during training:

<Math display expr="h_i^{\text{test}} = p \cdot h_i^{\text{no-dropout}}" />

**Inverted dropout** — the modern variant — moves the scaling to train time so the test forward pass needs no modification:

<Math display expr="\tilde{h}_i^{\text{inverted}} = \frac{r_i}{p} \cdot h_i" />

Now <Math expr="\mathbb{E}[\tilde{h}_i^{\text{inverted}}] = \frac{p}{p} h_i = h_i" />, matching the test activation exactly.

**Ensemble interpretation:** with <Math expr="n" /> hidden units, there are <Math expr="2^n" /> possible subnetworks (masks). Dropout weight-shares all <Math expr="2^n" /> models — the test-time network with scaled weights approximates averaging their predictions.

Implement **inverted dropout** in NumPy.

<HintFade client:load
  hints={[
    "You need a binary mask the same shape as x. Use np.random.rand and compare to keep_prob.",
    "np.random.rand(x.shape) < keep_prob gives a boolean mask — multiply x by it to zero out dropped units.",
    "After masking, divide by keep_prob (not multiply) to keep the expected activation equal to x.",
  ]}
  solutionCode={`import numpy as np

def dropout(x, keep_prob, rng=None):
    if rng is None:
        rng = np.random.default_rng()
    mask = rng.random(x.shape) < keep_prob
    return (x * mask) / keep_prob`}
/>
